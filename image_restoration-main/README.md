# Image Restoration for Surveillance Camera Systems

## Objective
This project simulates noise in surveillance camera systems and restores image quality using spatial filtering techniques.

## Noise Types
- Gaussian Noise (Sensor Noise)
- Salt & Pepper Noise (Transmission Errors)

## Filters Used
- Mean Filter
- Median Filter
- Gaussian Filter

## Performance Metrics
- Mean Squared Error (MSE)
- Peak Signal-to-Noise Ratio (PSNR)

## Observations

### Gaussian Noise
Gaussian filter performs best because it matches noise distribution.

### Salt & Pepper Noise
Median filter performs best as it removes impulse noise effectively.

## Conclusion
Different noise types require different filtering strategies for optimal restoration.
